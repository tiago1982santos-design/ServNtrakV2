2.1 CRIAR NOVO COMPONENTE WIZARD

Ficheiro NOVO: client/src/components/CreateClientWizard.tsx

Estrutura:
- 5 steps: Básico → Serviços → Faturação → Detalhes Técnicos → Localização
- Navegação com botões "Anterior" / "Próximo" / "Finalizar"
- Validação por step (não deixa avançar se step inválido)
- Progresso visual (1/5, 2/5, etc.)
- Dados persistem entre steps (useState central)

Step 1 - Informação Básica:
- name (obrigatório)
- phone
- email
- address

Step 2 - Serviços e Frequência:
- hasGarden, hasPool, hasJacuzzi (checkboxes)
- gardenVisitFrequency, poolVisitFrequency, jacuzziVisitFrequency (selects condicionais)
- serviceDurationMinutes

Step 3 - Faturação:
- billingType (radio: monthly, hourly, per_visit)
- monthlyRate / hourlyRate / perVisitRate (input condicional baseado em billingType)
- paymentMethod
- scheduledTransferDay (se monthly + transfer bancária)

Step 4 - Detalhes Técnicos (apenas se hasPool ou hasJacuzzi):
- Se hasPool: poolLength, poolWidth, poolMinDepth, poolMaxDepth
- Se hasJacuzzi: jacuzziLength, jacuzziWidth, jacuzziDepth

Step 5 - Localização:
- MapPicker (latitude, longitude)
- Mostrar resumo dos dados antes de submeter

Componente deve:
- Usar mesma validação (zodResolver com insertClientSchema)
- Usar mesmo hook useCreateClient
- Substituir CreateClientDialog.tsx COMPLETAMENTE

Ficheiro NOVO: client/src/lib/themes.ts

Conteúdo:
```typescript
export type ThemeName = 'verde' | 'azul' | 'laranja';

export interface ThemeColors {
  primary: string;
  primaryForeground: string;
  secondary: string;
  secondaryForeground: string;
  accent: string;
  accentForeground: string;
}

export const themes: Record<ThemeName, ThemeColors> = {
  verde: {
    primary: '152 55% 28%',        // Verde jardim
    primaryForeground: '0 0% 100%',
    secondary: '152 35% 92%',       // Verde claro
    secondaryForeground: '152 55% 25%',
    accent: '200 75% 50%',          // Azul
    accentForeground: '0 0% 100%',
  },
  azul: {
    primary: '221 83% 53%',         // Azul corporativo
    primaryForeground: '0 0% 100%',
    secondary: '214 95% 93%',       // Azul claro
    secondaryForeground: '221 83% 45%',
    accent: '142 76% 36%',          // Verde como accent
    accentForeground: '0 0% 100%',
  },
  laranja: {
    primary: '31 91% 51%',          // Laranja terra
    primaryForeground: '0 0% 100%',
    secondary: '48 96% 89%',        // Amarelo claro
    secondaryForeground: '31 91% 40%',
    accent: '199 89% 48%',          // Azul como accent
    accentForeground: '0 0% 100%',
  },
};

export function applyTheme(themeName: ThemeName) {
  const theme = themes[themeName];
  const root = document.documentElement;
  
  root.style.setProperty('--primary', theme.primary);
  root.style.setProperty('--primary-foreground', theme.primaryForeground);
  root.style.setProperty('--secondary', theme.secondary);
  root.style.setProperty('--secondary-foreground', theme.secondaryForeground);
  root.style.setProperty('--accent', theme.accent);
  root.style.setProperty('--accent-foreground', theme.accentForeground);
  
  // Salvar preferência em localStorage
  localStorage.setItem('servntrak-theme', themeName);
}

export function loadSavedTheme(): ThemeName {
  const saved = localStorage.getItem('servntrak-theme') as ThemeName;
  return saved && themes[saved] ? saved : 'verde';
}
